# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-06-02
### Added
- **JSON Export Engine:** Automated generation of structured `.json` reports optimized for direct LLM integration and API parsing.
- **Beautiful CLI UI:** Introduced a rich, color-coded terminal interface using native ANSI escape sequences.
- **Windows Launcher:** Added `start.bat` for one-click interactive execution without manual terminal commands.
### Changed
- Refactored core Python script for improved execution speed and cleaner visual feedback.

## [1.0.0] - Initial Release
### Added
- Core Python utility for fetching live repository metadata.
- Automated extraction of stars, forks, and open issues.
